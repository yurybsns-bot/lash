Инструкция по запуску

Вариант A — только браузер (без бэкенда):

  1. Получи API ключ на console.anthropic.com
  2. Вставь ключ в поле выше и заполни данные о мастере
  3. Всё — генерация работает прямо из браузера

Вариант B — с бэкендом (нужен для Telegram бота и Google Drive):

  1. Загрузи папку backend/ на Railway.app (бесплатно)
  2. Добавь переменные окружения из файла .env.example
  3. Скопируй URL из Railway и вставь выше
  4. Telegram бот включится автоматически
